# Holistic Lifestyle & Routine Tracker

**Author:** Suyash Kapse (25BCE10525)  
**Course:**  BYOP - VITyarthi  

##  Project Overview
The Holistic Lifestyle & Routine Tracker is a Python-based Command Line Interface (CLI) application designed to solve a common problem for university students: maintaining a balanced daily routine while navigating hostel life and standardized mess meals. 

Instead of tedious calorie counting, this tool focuses on holistic wellness by tracking core daily metrics—hydration, sleep, physical activity, and diet categories—and providing immediate, logically calculated feedback to help users balance their lifestyle.

##  Features
- **Daily Metric Logging:** Quickly input water intake, sleep hours, and daily activity.
- **Categorical Diet Tracking:** Simplifies food tracking by categorizing meals into Standard (Mess), Heavy/Junk, or Light/Healthy.
- **Instant Logic Feedback:** Uses conditional logic to provide immediate, actionable advice based on the day's inputs (e.g., flagging sleep deficits or recommending hydration).
- **Persistent Data Storage:** Automatically saves all daily logs to a cleanly formatted `routine_log.csv` file for long-term tracking.

##  Prerequisites
- Python 3.x installed on your system.
- Basic terminal or command prompt access.
- No external Python libraries are required (uses built-in `csv`, `os`, and `datetime` modules).
